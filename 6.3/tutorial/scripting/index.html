脚本编程

本手册中的所有示例及代码都使用 Bash 脚本实现，要求读者对 Bash 脚本语言有最基本的了解。Windows 用户若使用 Batch 脚本，则需要同时了解 Bash 和 Batch 的语法，并了解如何将本手册中的 Bash 脚本正确转换为 Batch 脚本。

这一节介绍本手册中或 GMT 使用过程中最常用的 Bash 和 Batch 语法，而不涉及不常用或更复杂的语法。
