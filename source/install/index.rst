安装
####

这一章介绍 GMT 的安装方法。可以直接安装 GMT 二进制包，也可以自行编译源码安装 GMT。
下面会分别介绍这两种安装方式，用户根据需求任选其一即可。对于大多数用户而言，建议安装二进制包。

不同操作系统下安装 GMT 二进制包：

   :maxdepth: 1
