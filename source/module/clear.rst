clear
=====

:官方文档: :doc:`gmt:clear`
:简介: 删除缓存目录、数据目录或会话目录，以及当前配置文件

语法
----

**gmt clear**
[ **all** | **cache** | **defaults** | **data** | **sessions** ]
[ |SYN_OPT-V| ]

可选选项
--------

**all**
    删除所有项目，包括缓存目录、数据目录、会话目录以及当前配置文件

**defaults**
    删除现代模式下当前会话的GMT配置文件gmt.conf

**cache**
    删除GMT缓存目录（默认为\ **~/.gmt/cache**\ ）及其内容

**data**
    删除GMT数据目录（默认为\ **~/.gmt/server**\ ）及其内容

**sessions**
    删除GMT会话目录（默认为\ **~/.gmt/sessions**\ ）及其内容

.. include:: explain_help_nopar.rst_

示例
----

清空GMT缓存目录::

    gmt clear cache

删除现代模式会话下的当前配置文件::

    gmt clear defaults
