velo
====

:官方文档: :doc:`gmt:supplements/geodesy/velo`
:简介: 在地图上绘制三种符号：矢量（向量）、十字线和旋转饼图

-S选项
------

**-S**
    ``-S`` 选项决定要绘制哪一种符号，及其输入数据的格式。

    **-Se**\ [*velscale*/]\ *confidence*\ [**+f**\ *font*]
    绘制矢量，并可选择用椭圆表示不确定度。输入数据中每列的含义为：

    #. 起点的经度
    #. 起点的纬度
    #. 矢量的东向分量
    #. 矢量的北向分量
    #. 东向分量的不确定度（1-sigma）
    #. 北向分量的不确定度（1-sigma）
    #. 东向分量和北向分量的correlation
    #. 名称（可选）

    说明：

    #. ``velscale`` 控制了矢量的缩放比例，其对应了速度值为1的箭头的长度，默认单位由参数
       :term:`PROJ_LENGTH_UNIT` 决定，也可以自己添加长度单位 ``i|c|p``
    #. ``confidence`` 设置了椭圆的2D置信区间的极限，比如0.95代表95%置信区间
    #. ``font`` 是名称的字体属性。默认值[9p,Helvetica,black]，使用 **+f**\ 0 则不显示名称。
    #. 椭圆的填充色由 ``-G`` 选项控制
    #. 椭圆的轮廓以及矢量的画笔属性由 ``-W`` 选项控

    **-Sr**\ [*velscale*/]\ *confidence*\ [**+f**\ *font*]
    与 ``-Se`` 类似，区别在于输入文件不同：

    #. 起点的经度
    #. 起点的纬度
    #. 矢量的东向分量
    #. 矢量的北向分量
    #. 椭圆的半长轴
    #. 椭圆的半短轴
    #. 椭圆的长轴相对于水平方向逆时针旋转的角度
    #. 名称（可选）

    **-Sn**\ [*barscale*]
    绘制各向异性矢量， ``barscale`` 是各向异性矢量的缩放比例。输入数据中每列的含义为：

    #. 经度
    #. 纬度
    #. 各向异性矢量的东向分量
    #. 各向异性矢量的北向分量

    **-Sw**\ [*wedgescale*/]\ *wedgemag*
    绘制旋转饼图。输入数据中每列的含义为：

    #. 经度
    #. 纬度
    #. 旋转饼图的旋转角度（弧度制）
    #. 旋转饼图的不确定度（弧度制）

    旋转饼图由两个同心扇形组成。一个扇形是旋转饼图本身（下称饼图扇形），另一个扇形则表示旋转饼图的不确定度（下称不确定度扇形）。
    两个扇形的半径由 ``wedgescale`` 给出。两个扇形的角度分别为旋转饼图的旋转角度和不确定度乘以 ``wedgemag`` 。
    饼图的填充色由 ``-G`` 选项控制， 不确定度的颜色由 ``-E`` 控制。
    饼图扇形从正北开始，顺时针旋转。不确定度扇形从旋转饼图的结束开始，向顺时针和逆时针两个方向旋转，两个方向旋转的角度均为不确定度的 2 倍。

    **-Sx**\ [*cross_scale*]
    绘制应变十字线。 ``cross_scale`` 控制十字线的大小。输入文件中每列的含义为：

    #. 经度
    #. 纬度
    #. eps1：应变张量的扩张本征值，扩张为正
    #. eps2： 应变张量的压缩本征值，扩张为正
    #. eps2的方位角（北方向顺时针旋转的角度）

选项
----

**-A**\ *parameters*
    修改矢量的属性。注意默认情况下，矢量头的大小不随着矢量线的长度变化而变化。
    当矢量线过短时，会出现矢量箭头不绘制的情况。此时可以添加 **+n**\ *norm* 属性让箭头大小随矢量成比例变化。
    详细说明见 :doc:`/basis/vector` 一节。

**-C**\ *cpt*
    提供一个CPT文件，并让通常由 ``-G`` 设置的符号颜色根据 magnitude 来确定。
    具体参见 ``-Z`` 。

**-D**\ *scale*
    控制 ``-Se`` 和 ``-Sr`` 中速度不确定度以及 ``-Sw`` 中旋转不确定度的缩放。该参数可以与 ``<confidence>`` 一起控制不确定度的绘制

**-E**\ *fill* 
    设置用于填充不确定性楔形图（ ``-Sw`` ）或速度误差椭圆（ ``-Se`` 或 ``-Sr`` ）的颜色或阴影。如果未指定 ``-E`` ，则不确定性区域将会是透明的。
    注意：使用 ``-C`` 和 ``-Z+e`` 将会根据 ``-Z`` 中选择的度量标准 [如 magnitude error ] 更新不确定性填充颜色。

**-G**\ *fill* 
    设置符号的填充色
    注意：使用 ``-C`` 和 ``-Z`` 会覆盖本选项。

**-L**\ [*pen*\ [**+c**\ [**f**\|\ **l**]]]
    绘制线段。椭圆和断层面会根据 ``-W`` 绘制出轮廓。
    使用 **+cl** 则会使用 ``-C`` 的设置使用CPT的颜色。
    使用 **+cf** 则只会将CPT设置应用在 error 误差填充上。
    使用 **+c** 则会将CPT设置应用在线条和填充上。

**-N**
    不跳过区域范围外的符号

**-W**\ [*pen*][**+c**\ [**f**\|\ **l**]]
    设置速度箭头、椭圆轮廓以及断层边界的颜色 [默认: width = 0.25p, color = black, style = solid]。
    使用 **+cl** 则会使用 ``-C`` 的设置使用CPT的颜色。
    使用 **+cf** 则只会将CPT设置应用在符号填充上。
    使用 **+c** 则会将CPT设置应用在线条和填充上。

**-Z**\ [**m**\|\ **e**\|\ **n**\|\ **u**\ ][**+e**]
    选择与 ``-C`` 一起使用的量，以设置填充颜色。可选择：
	- m：magnitude（矢量magnitude或旋转magnitude）
	- e：东西方向速度
	- n：南北方向速度
	- u：用户提供的数据列（位于必需列之后提供）

如果希望使用相应的误差估计值（即矢量或旋转的不确定性）来查找颜色并用于绘制误差椭圆或楔形图，请添加后缀 +e。
